## Sobre

Portfolio tecnico construido como produto, nao como template. Cada decisao de arquitetura, stack e UX tem justificativa documentada.

## Stack

| Camada | Tecnologias |
|--------|------------|
| Framework | Next.js 16, React 19, TypeScript |
| Styling | Tailwind CSS 4, Framer Motion |
| i18n | PT / EN com alternancia dinamica |
| SEO | JSON-LD, Open Graph, sitemap, feed RSS |
| Deploy | Vercel (Edge Network) |
| Qualidade | ESLint, WCAG 2.1 AA contrast compliance |

## Projetos

### BlackBelt v2
SaaS operacional para academias de artes marciais. Arquitetura serverless com Next.js, Supabase, Row Level Security e deploy na Vercel.

### Aura v1
Assistente operacional local-first com foco em privacidade e autonomia. Funciona offline, sem dependencia de servidores externos.

## Estrutura

```
app/           # App Router (pages, layouts, metadata)
components/    # Componentes React reutilizaveis
context/       # Providers (theme, i18n)
lib/           # Dados, utilidades, configuracoes
public/        # Assets estaticos
```
